bash脚本同常见编程语言类似，由一条条语句构成，但它是一行行直接执行的解释性语言，不需要进行编译。每行对应一条语句，语句结尾直接回车换行即可，不需要其他标志。

bash脚本由bash负责执行，脚本第一句#!/bin/bash用来指定脚本由bash执行。#。。。为注释行，#!不是注释，意义如前所述。
